# 💳 CreditWise Loan Prediction System

A Machine Learning project that predicts whether a loan will be approved based on applicant financial and credit-related information.

---

## 🚀 Project Overview

This system helps financial institutions assess loan approval decisions using historical data and machine learning techniques.

---

## 📊 Features

- Data preprocessing and cleaning  
- Feature engineering (log transformation, square features)  
- Model training using Machine Learning algorithms  
- Model evaluation using accuracy and confusion matrix  
- Predict loan approval (Approved / Rejected)

---

## 🧠 Machine Learning Model

- Algorithm: Logistic Regression  
- Evaluation Metrics:
  - Accuracy Score
  - Confusion Matrix
  - Precision & Recall

---

## 🛠️ Technologies Used

- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Jupyter Notebook
